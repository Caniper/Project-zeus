# Discord permission control software - ES6 - WIP

## Getting Started

1. Clone git repo
2. Run to install depencies:

```
npm install
```
and `cd front-end`
```
npm install
```

3. Run to start server:

in `/front-end`

```
npm run build
```

and in `/`

```
npm hot-start
```
